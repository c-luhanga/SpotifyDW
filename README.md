# SpotifyDW

Data warehouse project for Spotify data analysis using SQL Server and C# ETL.

## Structure
- `data/` - Source CSV files
- `db/` - Database schema definitions
- `src/SpotifyDW.ETL/` - C# ETL console application
- `docs/` - Project documentation

## Setup
1. Add CSV files to data/ folder:
   - spotify_data_clean.csv
   - track_data_final.csv
2. Execute schema.sql to create database
3. Run ETL application to load data
